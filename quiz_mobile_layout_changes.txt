  File to edit: src/components/quiz/UnifiedQuizPlayer.tsx

================================================================================

SCOPE:
  These changes apply ONLY to Cloze Test and Comprehension Test exam types
  (i.e., exams where the `passage` variable is non-empty).
  All other exam types (MCQ, Bilingual MCQ, Parajumble, Error Correction)
  remain completely unchanged.
  Desktop layout (lg: breakpoint and above) also remains unchanged.

GOAL:
  Portrait  → passage fills all available space at top (flex:1),
              question+options shrink to content size at bottom (flex-shrink:0),
              nav buttons always visible and fixed at the very bottom.
  Landscape → passage fixed at 48% width on the left (scrollable),
              question+options fill 52% on the right in one scrollable zone,
              nav buttons always visible and fixed at the bottom of the right panel.
  Minimum font size is never below 11px — content scrolls, font never shrinks further.
